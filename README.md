## Overview

FinApp is a web application made to help you manage your finances. It is an easy to use application that allows you to add your expenses, incomes, track them and a little more (check it yourself üëÄ).

FinApp is not available for anyone to use because it uses free tiers of some tools that I'm not able/interested to pay for. If you're interested on using it you can contact me and we can talk about it üôÇ.

## Main features

- Add expenses and incomes, called _transactions_ (unique or recurring)
- See month balance and pending transactions on the dashboard
- See insights about your transactions (soon)

## Techs, libs and tools used

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [Firebase (hosting)](https://firebase.google.com/)
- [Supabase (database, authentication, storage)](https://supabase.io/)
- [React Query](https://tanstack.com/query/latest/)
- [Chakra UI](https://chakra-ui.com/)
- [React Hook Form](https://react-hook-form.com/)
